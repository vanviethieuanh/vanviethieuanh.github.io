p5.js makes creative coding accessible to everyone. It's perfect for generative art, data visualization, and interactive experiences.

## Why p5.js?

- **Easy to Learn**: Intuitive API based on Processing
- **Browser-Based**: Runs anywhere with a web browser
- **Great Community**: Tons of examples and tutorials

## Simple Example

```javascript
function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  ellipse(mouseX, mouseY, 50, 50);
}
```
